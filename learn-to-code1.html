What I Learned

In this module, I explored the following programming concepts:

  • Variables: Named storage locations in memory used to hold data that can be changed during program execution.
  • Types: Classifications of data that define what kind of value a variable can hold, such as integers, strings, or booleans.
  • Initialization: The process of assigning an initial value to a variable at the time of its creation.
  • Parameters: Variables passed into functions to provide input values, allowing functions to operate on different data.
  • World Building: The practice of creating a structured environment or context in which programming concepts can be applied, often used in game development.
  • Arrays: Data structures that hold multiple values in a single variable, allowing for efficient storage and manipulation of lists of data.
