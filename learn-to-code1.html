What I Learned

In this course, I learned about the following programming concepts:

  • Commands: Instructions given to a computer to perform specific tasks, such as `print` in Python or `System.out.println` in Java.
  • For Loops: Control flow statements used to execute a block of code repeatedly for a fixed number of times, such as `for (int i = 0; i < 10; i++)` in Java.
  • Conditionals: Statements that allow the execution of code based on certain conditions, like `if`, `else if`, and `else` statements that execute different code blocks based on logical conditions.
  • Logical Operators: Symbols used to perform logical operations, such as `&&` (AND), `||` (OR), and `!` (NOT), to combine or invert boolean expressions.
  • Variables: Named storage locations in memory used to hold data that can be manipulated throughout a program, such as `int age = 25;` in many programming languages.
  • Types: Classification of data that specifies what kind of value a variable can hold, such as integers (`int`), floating-point numbers (`float`), or strings (`String`).
  • Initialization: The process of assigning an initial value to a variable at the time of its creation, like `int count = 0;` in programming.
  • Functions: Blocks of reusable code that perform specific tasks and can be called with a name, such as `void printMessage()` in Java or `def greet()` in Python.
  • Parameters: Variables passed into functions to provide input values that the function can use, such as `void add(int a, int b)` where `a` and `b` are parameters.
